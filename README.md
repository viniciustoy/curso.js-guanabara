### 🗂 Como baixar o projeto
```bash
# Clonar o repositório
$ git clone https://github.com/viniciustoy/curso.js-guanabara

# Entrar no diretório
$ cd curso.js-guanabara

# Instalar as dependências
$ npm install

# Iniciar o projeto
$ npm start
```
